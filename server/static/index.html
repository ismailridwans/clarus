Protocol
How It Works
Standard OpenEnv interface — three endpoints, deterministic grading, structured reward signal.
01
POST /reset
Start a new episode by choosing a task and optional seed. Receive the patient's complaint and case metadata.
02
POST /step
Execute actions — fetch records, write diagnoses, file resolutions. Each step returns a reward and updated observation.
03
close_case
When all evidence is gathered and the resolution filed, close the case. The grader runs 17–28 checks and emits a final score.
04
Score 0→1
Structural rewards guide exploration. Grader checks evaluate correctness — right party, right resolution, cited evidence.
